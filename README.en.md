# SceneV

#### Description
SceneV（Scene Visual）是一个强大的低代码数据可视化开发平台，专为工业组态和数据可视化大屏设计而打造。它让用户能够通过直观的拖拽操作，零代码或低代码方式快速构建专业级的可视化应用。它的技术栈为：Vue3 + TypeScript + Vite + Element Pluse+ ECharts
